# **JEC**

A **Hypixel SkyBlock** mod, developed by **[me](https://github.com/skies-starred)**.

## Features

- Auto meow: automatically sends one of the selected keyword randomly when one of the selected keywords is detected in a message.
- Catify messages: catifies messages that you send!
- Cat facts: randomly sends cat facts in chat. Adds "/jec fact" as well.
- Cat models: turn yourself, npcs, other players into cats. Supports baby versions and custom cat textures!
- Chat sounds: plays a cat sound when someone sends one of the selected words in chat.
- Kill sounds: plays a cat sound when someone kills a mob.
